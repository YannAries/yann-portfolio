Après avoir obtenu mon diplôme pré-universitaire au Collège Ahuntsic à Montréal au Canada et décroché un stage de 3 mois chez Libéo, j'ai réalisé que mon éducation formelle m'a donné la théorie, mais les périodes de stage et l'autoformation m'ont apporté des compétences qui comptent en milieu professionnel.

### 1. Apprentissage axé sur les projets

**Ce qui fonctionne plutỗt bien pour moi :** Choisir un projet, apprendre ce dont j'ai besoin quand j'ai en ai besoin.

**Exemple :**
- **Objectif :** Construire un site portfolio
- **Appris et perfectionné :** Hugo, Git, Docker, déploiement
- **Résultat :** Vous le voyez

**Cela m'a motivé à apprendre :**
- Générateurs de sites statiques
- Infrastructure CDN (Bunny Fonts, jsDelivr)
- Optimisation d'icônes (Simple Icons vs Font Awesome)
- Débogage de performance

Toutes des compétences que j'utilise maintenant dans divers projets.

### 2. Documentation plutôt que cours

**Ma meilleure approche :**
1. Lire d'abord la documentation officielle
2. Construire quelque chose de petit
3. Le casser, le réparer, comprendre pourquoi
4. Passer à la fonctionnalité suivante

**Mes liens favoris :**
- [Hugo Docs](https://gohugo.io/documentation/) - Claire, recherchable, actuelle
- [MDN Web Docs](https://developer.mozilla.org/) - La bible du développement web
- [PostgreSQL Docs](https://www.postgresql.org/docs/) - Meilleure que n'importe quel tutoriel

### 3. Prioriser les tâches avec la règle 80/20

Pas besoin de tout maîtriser, juste se concentrer sur les 20% qui donnent 80% des résultats.

**Mon stack principale :**
```
Essentiel :
├── Python
├── HTML5, CSS3, JavaScript avancé
├── PostgreSQL
├── Git
└── Docker

Bon à avoir :
├── React ou Vue
├── SSGs (Hugo)
├── Redis
├── IA/Cloud computing
└── CI/CD
```

## Ressources que j'utilise

### Pour le développement web

1. [Developpez.com](https://www.developpez.com/)

    Communauté française de développeurs avec articles sur l'actualité tech et tendances,<br> tutoriels et forums

2. [Learnpython.org](https://learnpython.org/)

    Cours basiques et avancés

    Basé sur la résolution de problème

3. [MDN Web Docs](https://developer.mozilla.org/en-US/)

    Utilisation en construisant des projets

    Rechercher au lieu de lire de manière linéaire

### Pour l'analyse de données

1. [Kaggle Learn Courses](https://www.kaggle.com/learn)

    Cours pratiques de data science et machine learning

    Python, SQL, IA

2. [Plotly](https://plotly.com/python/)

    Documentation sur la création de graphiques interactifs, cartes, dashboards<br> et visualisations scientifiques en Python

3. [eBook - Python for Data Analysis by Wes McKinney](https://wesmckinney.com/book/)

    Le guide de référence pour pandas
